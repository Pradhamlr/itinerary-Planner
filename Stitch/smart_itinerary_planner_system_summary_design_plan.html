# Smart Itinerary Planner: Current System Summary ## 1. Project Overview A full-stack Smart Itinerary Planner with React frontend, Node.js/Express/MongoDB backend, and Python FastAPI ML inference layer. Supports: auth, trip creation, place recommendation (ML-assisted), day-wise itinerary generation, map visualization, editable itineraries. ## 2. Key UI Components to Design - **Trip Creation:** Location autocomplete, date selection, budget, interests (beaches, shopping, culture, etc.). - **Recommendations:** Display attractions, restaurants, ML scores, and swap/refresh options. - **Itinerary View:** Day-wise clusters, time slots, travel pacing, map integration (Google Maps). - **Interactive Controls:** Lock places, swap/replace from hidden pool, drag-drop reorder. ## 3. Brand & Style Goals - Modern, travel-focused aesthetic. - Data-rich but clean (handling ML scores and complex schedules). - High-quality map integration.
